Airtable RSVP Setup

Follow these steps once — it only takes about 5 minutes and responses will flow into a tidy spreadsheet automatically.

  1. Go to airtable.com and create a free account.
  2. Create a new Base — name it anything (e.g. Wedding RSVPs).
  3. Inside, rename the default table to RSVPs and add these fields:
    Name  Attendance  Party Size  Message
  4. Go to airtable.com/create/tokens → create a Personal Access Token with scope data.records:write and access to your base.
  5. Find your Base ID: visit airtable.com/api, select your base — the Base ID (starts with app…) is shown in the URL and docs page.
  6. Open index.html in any text editor. Find the RSVP CONFIG block near the bottom of the file and replace YOUR_TOKEN_HERE and YOUR_BASE_ID_HERE with your values.
  7. Save & done — every RSVP submission will appear in your Airtable base in real time 🌿
